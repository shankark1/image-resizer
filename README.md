# Modern Image Resizer

A sleek, user-friendly, browser-based image resizing application built with modern web technologies. This tool allows users to quickly resize images without needing any server-side processing, ensuring complete privacy and fast results.

## Features
- **Client-Side Processing**: All image resizing is done directly in the browser using the HTML5 Canvas API. No images are uploaded to any server.
- **Drag & Drop Support**: Intuitive interface allowing users to drag and drop images directly into the upload zone.
- **Aspect Ratio Lock**: Maintain the original proportions of your image easily by toggling the aspect ratio lock.
- **Quick Size Presets**: One-click resizing to common dimensions (1920x1080, 1080x1080, 1280x720, 800x600).
- **Modern UI/UX**: Clean, responsive design featuring glassmorphism elements, dynamic background blobs, and smooth micro-animations.
- **Format Support**: Works seamlessly with JPG, PNG, and WEBP formats.

## Technologies Used
- HTML5
- CSS3 (Vanilla, custom properties, flexbox/grid)
- JavaScript (Vanilla, ES6+)
- Phosphor Icons (for modern iconography)
- Google Fonts (Outfit)

## Getting Started
Since this is a fully static web application, no build steps or local servers are required to run it.

1. Clone or download this repository.
2. Open the `image-resizer` directory.
3. Double-click `index.html` to open it in your default web browser.
4. Drag and drop an image to start resizing!

## Usage
1. Click the upload zone or drag and drop an image file into it.
2. Enter your desired custom width and height, or click on one of the quick size presets.
3. Toggle the link icon between the width and height inputs to lock or unlock the image's aspect ratio.
4. Click the "Download Image" button to save your resized image.
5. Click "Reset" to clear the current image and start over.
